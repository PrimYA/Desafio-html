💥 Estrutura Básica de HTML

A linguagem de marcação HTML é focada na estrutura e no conteúdo de uma página web. Para isso, usamos as marcações < > (No inglês, “tags”).


🖊 Falando Sobre Tags

Voltar

Cada tag tem sua finalidade e também hierarquia, no caso dos títulos e subtítulos, por exemplo. Para usá-las, é preciso iniciar o documento HTML. Normalmente, o "esqueleto" de um site começa assim:

Base de código HTML 
        
          <!DOCTYPE html>
          <html lang="pt-BR">
          <head>
          <meta charset="utf-8">
          <title></title>
          </head>
          <body>
          </body>
          </html>
“Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web”. (MDN Web Docs)

🔖 Nome das Tags

Voltar

< !DOCTYPE html >

Esta é uma tag obrigatória para iniciar um arquivo HTML. Por convenção, essa linha de código indica que o documento está, ou será escrito, em HTML5 (Versão mais atualizada).

<html> </html>

É este elemento que deve englobar todo conteúdo do site. Com exceção da tag “Doctype” , se alguma outra estiver fora, isso significa que não será lida. Por isso, chamamos esta marcação de elemento raiz do HTML (root element).

O idioma de uma página web pode ser indicado dentro da tag, como em: < html lang="pt-BR" >, ou <html lang="en">.

<head> </head>

Este elemento contém, em seu corpo, algumas configurações padrão e adicionais que o autor pode incluir (Gera informação adicional no mecanismo de busca e não será visível na página.), como:

<meta charset="utf-8">

Utilizado para definir o conjunto de caracteres usados no arquivo, sendo possível que qualquer máquina ou pessoa tenha acesso seguro aos dados deste site.

<title> </title>

Dentro desta tag deve conter o título da página web. Ele será mostrado na aba do navegador.

A marcação <head> também pode ter link para folhas de estilos (css), formato do site e componentes externos, como bibliotecas e outras dependências.

<body> </body>

O “body”(corpo, em inglês), é o que vem depois da head. Nele, inserimos outras tags que irão ajudar a organizar todo tipo de informação que queremos, como mídia, textos, campos de preenchimento, botões, listas, links, etc.

🖌 Atributos das Tags

Voltar

Os atributos são inseridos dentro das tags, como o exemplo do “class” , que identifica um ou vários elementos de mesma classe . O atributo class é muito usado para conectar elementos às folhas de estilo (css).

📰 Adicionando Textos

Voltar

Antes dos textos, vêm os títulos e subtítulos (Headings). Eles têm seis níveis e servem para organizar o conteúdo do site.
